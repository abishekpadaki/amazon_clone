# Amazon Shopping - Clone using ReactJs and Firebase

**Project Link** - ***https://abiks-amazon-clone.vercel.app/***

## Technologies Used

- ReactJs
- Firebase Hosting
- Firebase Auth
- Firebase Storage
- React-Dom
- React Redux
- Stripe payment

## Features

- Login and Logout
- Add to cart
- Remove from cart
- Payment 
- Neat and clean UI

## Steps to run in your machine

#### Run the following commands 
```
npm i
npm run start
```
